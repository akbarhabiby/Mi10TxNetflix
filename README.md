# Mi10T x Netflix

## Project setup
```
npm install
```

## Run the bot
```
npm start
```

## Run the bot via custom IMEI prefix
```
node app.js xxxx
```
> xxxx is your IMEI prefix

### Supported region
- MY (Malaysia)
- TH (Thailand)
- PH (Philippines)
